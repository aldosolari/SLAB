Laboratorio di Statistica per l’Economia

Università Ca’ Foscari di Venezia, Corso di Laurea triennale in Economia e Commercio.

Esame

Agli studenti sarà richiesto di analizzare un insieme di dati per verificare la loro capacità di analizzare, interpretare e comunicare le informazioni.

Contenuti

  1. Introduzione ad R.
  2. Statistica descrittiva.
  3. Calcolo delle probabilità.
  4. Inferenza statistica.
  5. Analisi dei dati.

Materiale didattico

Il software R

  • R è un software open source ed è possibile scaricarlo al link https://cloud.r-project.org.

  • L’uso di Rstudio è facoltativo in questo corso, anche se fortemente consigliato. È possibile scaricarlo al link https://rstudio.com/products/rstudio/.

  • La documentazione del software R è parte integrante del materiale didattico.

Dati

  • Titanic

French Lottery Data

Due file contenenti dati della lotteria francese dal 1758 al 1834. I dati sono in formato testo. Il primo file (“Winning Draws”) contiene 6606 righe, ognuna con cinque numeri separati da tabulazioni. Il secondo file (“Winning Quaterne Bets”) ha 327 righe, ognuna con quattro numeri, separati da tabulazioni. Non ci sono etichette o righe di intestazione.
